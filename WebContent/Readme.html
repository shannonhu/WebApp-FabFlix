How to use the parser
  1. Go to the fabflix/src directory
  2. Compile "StartParsing.java" using class path "../mysql-connector-java-5.1.34-bin.jar:."
  3. Run "StartParsing" using class path "../mysql-connector-java-5.1.34-bin.jar:."
  4. compile_run.sh is located in fabflix root directory
  5. The first run will be slower following exam will be faster because no more data is added into database.
